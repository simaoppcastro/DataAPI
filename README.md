## Introdução
* Data Enterprise Gateway
* .NET Core (2.1) + Angular (9)
* API para gestão de dados de empresas 

## Objectivos
Desenvolver uma API com o propósito de oferecer uma plataforma de gestão de dados empresariais 

* dataAPI
    * Administrador
        * Gestão de utilizadores
        * Gestão de dados
        * ClientsData
        * DataCharts
        * Servers
    * Utilizador
        * Aceder a dados
            * DataCharts
            * Servers
